Notacion del cubo

Este dodecaedro al tener varias caras tiene una notacion mas enrevesada ya que hay dos tipos de notacion, una para cuando tienes que deshacer el cubo y otra para cuando tiens que resolverlo

Esta es la Notacion para el Scramble o para deshacer el cubo

  • U
  • D
  • R

Para Diferenciar si el movimiento es horario o antihorario, se usara el ++ o -- despues de la letra en caso de que sea R o D, Pero en caso de que sea la U se hara lo mismo que en los otros utilizando " ' "

Esta es la notacion para Solucionarlo

  • U
  • R
  • L
  • F
